## 📋 Sobre el Proyecto

El objetivo de esta web es analizar el impacto del hardware en nuestro planeta y buscar soluciones sostenibles. Los temas principales que tratamos son:

1. **Contaminación ambiental:** Causas, tipos y la nueva contaminación invisible.
2. **Residuos informáticos:** El problema de los metales pesados y los vertederos tecnológicos.
3. **Obsolescencia programada:** Estrategias de la industria para acortar la vida útil de los dispositivos.
4. **Informática ecológica:** Soluciones como el reciclaje, el bajo consumo y la minería urbana.
